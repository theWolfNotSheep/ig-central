# Google Drive Watch Folders — Demo Script

**Duration: 2:30**

**Goal:** Show how records managers can designate Google Drive folders for automatic, ongoing classification — so new documents are governed the moment they land.

## Script

1. **Open with the daily reality.** *"Your team drops new documents into Google Drive folders every day. Employment contracts into 'HR Onboarding', invoices into 'Finance Inbox', case files into 'Legal Matters'. One-time classification is useful — but you really want new arrivals classified automatically, every time."*

2. **Navigate to Drives → Google Drive** (a connected account).

3. **Browse to a folder.** Pick a representative one — e.g. HR Onboarding.

4. **Click 'Watch Folder'.** *"Three options — watch only new files, watch all files including existing, or watch and re-classify periodically. Pick new-only for most cases."*

5. **Configure the watch.** *"Set the classification pipeline — default or a category-specific one. Set the sensitivity ceiling — 'nothing higher than CONFIDENTIAL from this folder without review'. Save."*

6. **Demonstrate a new file arriving.** Upload a file into the watched folder from Drive. *"Within seconds, the watcher picks up the change. The file enters the pipeline — text extracted, PII scanned, classified, governance applied. No human action required."*

7. **Navigate to monitoring.** *"The pipeline activity feed shows the new file flowing through in real time — same four stages, same audit trail as an uploaded document."*

8. **Show the Watched Folders view.** *"See every folder being watched — source folder, watch mode, pipeline, file count processed, any errors. Pause, resume, or stop watching with one click."*

9. **Tie to the user workflow.** *"Your team keeps working the way they already work — dropping files into Drive. IG Central just makes sure every one of them ends up classified, governed, and retrievable."*

**Key message:** *"Governance by default, invisible to users. The moment a document lands in Drive, it's classified and compliant — no change to your team's habits."*
