# Audit Trail — Demo Script

**Duration: 2:30**

**Goal:** Show the comprehensive audit log — every user action, every policy change, every document access — ready for compliance queries and regulatory evidence.

## Script

1. **Open with the question every auditor asks.** *"In every ISO 27001, SOC 2, or ICO audit, the first thing the auditor says is 'show me your logs'. You need an answer that isn't a pile of CSV files. Let me show you ours."*

2. **Navigate to Admin → Audit Trail.**

3. **Show the event timeline.** *"Every event captured — user logins, document accesses, classification changes, policy updates, retention schedule edits, pack imports, permission changes, API calls. Timestamp, actor, action, target, outcome."*

4. **Filter by actor.** *"Want to know everything a specific user did last quarter? Filter by user, pick a date range. Every action they took is there — documents viewed, classifications overridden, exports taken."*

5. **Filter by target.** *"Want to know everyone who touched a specific document? Filter by document ID or slug. You get the complete access history — who, when, what they did."*

6. **Filter by event type.** *"Filter by event type — 'access denied' events tell you who tried to reach something they shouldn't have. 'export' events tell you where data has left the platform. 'policy change' events tell you who altered governance."*

7. **Show an entry in detail.** *"Expand any entry — you see the before/after state, the IP address, the session ID, the request trace. Forensic level of detail when you need it."*

8. **Show retention controls.** *"Audit events have their own retention — configurable per category. GDPR says log data retention should be justified. You set it per event type — seven years for classification changes, one year for login events."*

9. **Export.** *"Export by filter as CSV or JSON. Your auditor gets exactly the evidence they requested, formatted for their tooling."*

**Key message:** *"Audit isn't an afterthought — it's the substrate. Any question your auditor asks, you answer with a filter and an export."*
